              Upgrade this drive to v2?

              This drive was set up with YTDisc v1. v2 introduces accounts, music, and other features that require a new on-disk layout (a .data folder next to Videos and Music).

              The upgrade is one-way. If you'd rather keep this drive compatible with v1, quit and use the v1 binary instead — it still works fine.
